    Ellabib API v1.0.0

    This is the API for the Ellabib application

    Base URLs:

    • https://api.ellabib.se

    User

    Everything about the authenticated user

    getUser

    Code samples

    $.ajax({
  url: 'http://api.ellabib.se/user/id/:id',
  method: 'get',
})

    GET /user

    Get user info

    Example response

    {
    "data": [
        {
            "id": 1,
            "firstName": "Gabriel",
            "lastName": "Wallén",
            "extId": null,
            "roleType": "admin",
            "roleDisplayName": "Administratör",
            "pagesRead": "155",
            "booksRead": 1,
            "reviewsWritten": "1",
            "classDisplayName": "1A",
            "classId": 1,
            "avatarId": 1,
            "avatarImageUrl": "https://ellabib.se/images/avatar-1.png",
            "avatarDisplayName": "basic",
            "avatarType": "avatar-1"
        }
    ]
}

    Responses

    Status Meaning Description Schema
    200 OK succesful operation Inline
    405 Method Not Allowed Invalid input None

    Response Schema

    Status code 200

    Name Type Required Description
    id integer true Unique ID for user
